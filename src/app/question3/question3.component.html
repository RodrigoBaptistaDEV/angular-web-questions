Questão 3

Descubra a lógica e complete o próximo elemento:
a ) 1, 3, 5, 7, ___
b ) 2, 4, 8, 16, 32, 64, ____
c ) 0, 1, 4, 9, 16, 25, 36, ____
d ) 4, 16, 36, 64, ____
e ) 1, 1, 2, 3, 5, 8, ____
f ) 2,10, 12, 16, 17, 18, 19, ____

a) O próximo elemento é 9 pois cada elemento surge pela soma do anterior com 2.
b) O próximo elemento é 128 pois cada elemento surge pela multiplicação do anterior por 2.
c) O proximo elemento é 49 (quadrado de 7) pois os elementos são os quadrados da sequência 0,1,2,3,4,5,6.
d) O proximo elemento é 100 (quadrado de 10) pois os elementos são os quadrados da sequência 2,4,6,8.
e) O próximo elemento é 13 pois cada cada elemento surge da soma dos dois anteriores.
f) O próximo elemento é Dozentos (200) pois é uma sequência de números que começam com a letra D.
